## Основний проект: Notes & Wishboard Hub

Це підсумкова робота, яка інтегрує всі вивчені технології в єдину систему:

- **Spring Boot API** (Порт `9001`): Сервіс для керування нотатками.
- **Dropwizard Analytics** (Порт `9002`): Модуль аналітики та дошки бажань.
- **Legacy Servlets** (Порт `9000`): Сервіс автентифікації та профілів.
- **Frontend**: Клієнтська частина на Vanilla JS.
